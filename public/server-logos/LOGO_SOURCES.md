## Quick Setup Instructions

1. Download logos from the sources above
2. Convert to SVG format if needed (using tools like Inkscape, Figma, or online converters)
3. Optimize SVG files (remove unnecessary metadata, optimize paths)
4. Place files in `public/server-logos/` directory
5. Ensure filenames match exactly:
   - `langchain-mcp.svg`
   - `google-maps.svg`
   - `google-workspace.svg`
   - `brave-search.svg`
   - `puppeteer.svg`
   - `filesystem.svg`
   - `postgres.svg`

## Logo Requirements

- **Format**: SVG (preferred) or PNG
- **Size**: Optimized for 32x32px to 64x64px display
- **Colors**: Should work on both light and dark backgrounds
- **License**: Ensure you have rights to use the logos (most open source projects allow logo usage)
